# Contributing

Thanks for helping improve Tape Calculator.

## Development

1. Use JDK 17 and Android SDK 36.
2. Keep the app offline-first and dependency-light.
3. Run `testDebugUnitTest` and `assembleDebug` before submitting a change.
4. Keep user-facing Czech copy clear and concise; English localization is planned.

## Scope

This project intentionally stays focused on basic arithmetic, an interactive tape, and five memories. Large scientific-calculator feature sets are out of scope unless discussed first.

## Privacy

Do not add analytics, advertising SDKs, accounts, network access, or telemetry without a public design discussion and explicit opt-in behavior.
